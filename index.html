Freedom E SDK¶

This is the documentation for the SiFive Freedom E SDK v20.05.RC.04.

Freedom E SDK is generally available from the Freedom E SDK Repository.

What is Freedom E SDK?¶

Freedom E SDK is a project maintained by Sifive Inc. which makes it easy to get started developing software for the Freedom E and Freedom S Embedded RISC-V Platforms.
